Installation


curl http://laravel.com/laravel.phar > laravel.phar
sudo mv laravel.phar /usr/local/bin/laravel
sudo chmod 777 /usr/local/bin/laravel

Create a new project


cd ~/Sites/
laravel new projectname
cd projectname

Initial permissions

                                
chmod -R 777 app/storage
